¡Hola! Soy un apasionado desarrollador front-end con experiencia en la creación de sitios web atractivos y funcionales. Mi enfoque principal es transformar diseños en código limpio y eficiente, asegurando una experiencia de usuario excepcional en todos los dispositivos.

Experiencia:

  • Más de 1 año de experiencia en el desarrollo front-end, trabajando en proyectos variados desde sitios web para empresas locales hasta aplicaciones interactivas.
  • Amplio conocimiento en la implementación de diseños responsivos para garantizar que los sitios se vean y funcionen de manera óptima en diferentes dispositivos y tamaños de pantalla.

Habilidades Técnicas:

  • Colaboré en el desarrollo de un sitio web de comercio electrónico, optimizando la velocidad de carga y mejorando la tasa de conversión en un 15%.
  • Implementé un sistema de animaciones y transiciones utilizando CSS y JavaScript, lo que mejoró la experiencia visual del usuario y aumentó el tiempo en el sitio.
  • Trabajé en un equipo multidisciplinario para crear una aplicación web interactiva que recibió elogios por su diseño y usabilidad.

Educación:

  • Título en Desarrollo Web Front-End, Platzi.
  • Cursos en línea y autodidacta para mantenerme actualizado con las últimas tendencias y tecnologías.
